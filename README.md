# Password Generator in C

This project is a simple password generator written in C.  
It generates random, secure passwords based on user-defined length.

## Features
- Generates random passwords
- Uses uppercase, lowercase, numbers, and special characters
- User-defined password length

## How to Run
1. Compile the program using a C compiler
2. Run the executable and enter the desired password length

## Technologies Used
- C programming language
- Standard C libraries
